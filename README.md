# 头发秀秀 - AI发质分析

一款基于AI图像识别的头发/头皮分析H5网页应用。

## 功能特性

- 📸 拍照或相册选择照片
- 🔬 AI智能分析发质类型
- 📊 头皮健康状况评估
- 💡 个性化护理建议
- 📱 手机端优化，扫码即用

## 使用方法

1. 扫描二维码或访问链接
2. 点击"开始检测"
3. 拍摄或上传头发照片
4. 等待AI分析
5. 查看分析结果和护理建议

## 技术栈

- HTML5 + CSS3 + JavaScript
- 响应式设计，适配移动端
- 模拟AI分析（当前版本）
